⬆ 匯入資料

貼上 CSV 或 Tab 分隔的資料（格式：站點名稱、起始日期、Due Date、進度%、備註），每列一筆，不需標題列。

支援逗號（,）或 Tab 分隔。日期格式：YYYY/MM/DD 或 YYYY-MM-DD。

🔄 更新假日資料

美國、台灣、中國假日均可從公開 API 自動抓取，資料將儲存至 localStorage，下次開啟自動讀取。

選擇要抓取的年份與地區，點擊「開始抓取」即可自動從公開 API 抓取假日並儲存。台灣假日來自 ruyut/TaiwanCalendar，美國、中國來自 Nager.Date API。

地區 資料來源 更新方式 資料範圍
🇺🇸 美國 Nager.Date API（免費，無需 Key） 自動抴取 可抴取任意年份
🇨🇳 中國 Nager.Date API（免費，無需 Key） 自動抴取 可抴取任意年份（包含春節連假展開）
🇹🇼 台灣 ruyut/TaiwanCalendar（GitHub，免費） 自動抓取 可抓取任意年份（含補假、調休日）
